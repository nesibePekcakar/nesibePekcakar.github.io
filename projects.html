AI-Powered Error Log Analyzer

Developed an AI-powered web system that analyzes and explains software error logs. Fine-tuned multiple Mistral-7B and TinyLLaMA models using custom datasets in both English and Turkish, implemented the backend in .NET and built a full connection pipeline to interact with the models in real time.

The project integrates multilingual understanding, intelligent debugging suggestions, and an optimized model inference tunnel for seamless AI communication.

    Key Modules:
  • Mistral Error Helper (EN): Fine-tuned model for English debugging explanations.
  • Mistral Turkish Error Helper v2: Turkish variant optimized for localized error interpretation.
  • Mistral-7B Error Solver: Early prototype with multilingual reasoning capability.
  • Slidified LLaMA Error Dataset: Custom dataset built from structured error-solution pairs for model fine-tuning.

Technologies: Python · .NET · Blazor · Hugging Face · Mistral-7B · TinyLLaMA · PEFT · Google Colab · wandb · JSONL

AIPrompt – Personalized Prompt Optimization System

Developed an intelligent web system that personalizes prompt generation for university-level Computer Engineering students. The system uses user profiles — including topic proficiency, English level, and preferred response format — to optimize how large language models (LLMs) answer questions in a personalized and comprehensible way.

Trained using custom-built datasets and integrated with multiple AI models (ChatGPT, DeepSeek, Claude), AIPrompt enables adaptive learning experiences where students receive explanations at the level they can best understand.

Technologies: Python · HTML · OpenAI API · Flask · JavaScript · Prompt Engineering · Dataset Curation
